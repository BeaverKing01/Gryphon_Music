3 Chord Trick

Take any scale – notes, 1, 4, 5 make up the 3 chord trick

 

C maj

 

C    D     E    F    G    A    B     C

G:               G       C       D

D:               D       G       A

A:               A       D       E

E:               E       A       B

B:               B       E       F#

F#:             F#     B       C#

Relative Minor

Has the same notes as it’s major counterpart, but in a different order.

The 6th note is the relative minor starting point

Therefore if we take C – the 6th note is A. Therefore A minor is the relative minor to C.

C MAJOR -    C         D         E          F          G         A         B         C
A MINOR -    A         B         C         D         E          F          G         A        

A MAJOR-     A         B         C#       D         E          F#        G#       A
